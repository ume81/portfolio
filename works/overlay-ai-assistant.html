AI Android // 02

オーバーレイ
AIアシスタント

どのアプリを使っていても画面上に常駐するLive2DキャラクターがAIアシスタントとして機能するAndroidアプリ。タップして話しかければLLMが応答し、タスク管理やアプリ起動も自然な会話で操作できる。

demo / screenshot
overview
PROBLEM
タスク管理アプリを別途開く手間が続かない。AIに相談したくても専用アプリへの切り替えが面倒。作業中に画面から手を離さず操作したい
SOLUTION
Live2DキャラクターがAndroidオーバーレイとして常駐。どのアプリを使っていてもタップ一つで会話でき、タスク登録・通知・アプリ起動まで画面切り替え不要で操作できる
TARGET USER
AIアシスタントを日常使いしたいAndroidユーザー。スマホ操作中に手軽にタスクを登録したい人。キャラクターとのインタラクションを楽しみながら生産性を上げたい人
PLATFORM
Android専用（Flutter製）。TYPE_APPLICATION_OVERLAY権限を使った全画面透過オーバーレイ実装。Google Play非公開・APK直接配布
features
🧍
Live2D常駐オーバーレイ
WebView内でPIXI.js + pixi-live2d-displayを使いLive2Dキャラクターを描画。Kotlinネイティブの独自実装（flutter_overlay_windowパッケージ不使用）で全画面透過オーバーレイを実現し、他のアプリ使用中も常に表示される。
💬
LLM会話（Claude / Gemini API）
タップで会話パネルを開いてテキスト入力するとClaude APIまたはGemini APIが応答。設定でモデルを切り替え可能。会話履歴はHiveでローカル保存し、文脈を保ちながら連続して会話できる。
📋
タスク管理・期限通知
ToDoリスト・買い物リストをオーバーレイから直接登録。期限が近づくとキャラクターが通知し、スヌーズ（5分/15分/30分）にも対応。タスクデータはHiveでオフライン保存するためネット不要で動作。
🎭
キャラクター性格カスタマイズ
性別・性格タイプ・詳細パラメータの3段階でAIアシスタントの返答スタイルをカスタマイズ。設定した性格はシステムプロンプトに反映され、会話のトーン・言葉遣いが変化する。
🚀
アプリランチャー
よく使うアプリをお気に入り登録してオーバーレイから直接起動。カテゴリ分類・キーワード検索・インストール済みアプリの自動更新に対応し、ホームに戻らずアプリを切り替えられる。
🖼️
画像キャラクター・アニメーション
Live2Dモデル（.model3.json形式）に加え、静止画キャラクターも選択可能。待機・話す・うなずく・驚く・喜ぶの5種類のアニメーションをバッテリー消費を抑えながら再生。
tech stack
Flutter 3.16+ / Dart Kotlin（独自オーバーレイ実装） Live2D Cubism 3.x/4.x Claude API / Gemini API PIXI.js + pixi-live2d-display flutter_inappwebview Provider（状態管理） Hive（ローカルDB） flutter_secure_storage（APIキー） shared_preferences
technical notes

オーバーレイ実装はflutter_overlay_windowパッケージを使わず、Kotlinで完全独自実装した（Live2DOverlayService.kt）。TYPE_APPLICATION_OVERLAY権限で全画面透過Windowを生成し、キャラクター以外の領域はタッチイベントをスルーすることで下のアプリを邪魔しない設計にした。

Live2DはKotlinオーバーレイサービス内のWebViewでPIXI.js + pixi-live2d-displayを使って描画。WebView ↔ Flutterメインアプリ間をJavaScriptチャネルで繋ぎ、アニメーション切り替えや表情変化のトリガーをFlutter側から制御している。待機中はアニメーション更新間隔を伸ばしてバッテリー消費を最小化。表示キャラクターはLive2D公式サンプルモデルを使用している。

Claude APIとGemini APIをプラグイン式に切り替え可能な設計にし、APIキーはflutter_secure_storageで端末内に暗号化保存。タスク・チャット履歴はHiveでオフライン保存するため、LLM会話以外の機能はネット不要で動作する。
