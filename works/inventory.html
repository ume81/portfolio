Web App // 06

バーコード
在庫管理アプリ

商品バーコードをスキャンするだけで在庫の入出庫を管理できるWebアプリ。小売・飲食・EC系の個人事業主や小規模店舗向けに、シンプルな操作性と実用的な機能を両立した。

demo / screenshot
overview
PROBLEM
手書き・Excelでの在庫管理は手間がかかりミスが起きやすい。高機能な在庫管理システムは個人事業主には過剰で高コスト
SOLUTION
バーコードをスキャンするだけで入出庫を記録。スマホ1台あれば使えるシンプルな在庫管理を実現
TARGET USER
美容院・理容院向けのマルチテナントSaaS。テナントオーナー・スタッフ・管理者の3ロール構成
PLATFORM
Webアプリ（スマホのカメラでバーコードスキャン対応）
features
📷
バーコードスキャンで即記録
スマホカメラでJAN・QRコードをスキャンするだけで商品を特定し、入庫・出庫を記録。手入力不要。
📊
リアルタイム在庫一覧
商品ごとの現在在庫数をリアルタイムで確認。検索・カテゴリフィルタで素早く目的の商品を特定。
🔔
在庫アラート
設定した閾値を下回るとアラート通知。発注忘れを防ぐ。
📋
入出庫履歴・CSV出力
すべての入出庫履歴を記録。期間指定でCSVエクスポートし会計・棚卸しに活用できる。
tech stack
Laravel 11 / PHP 8.2 React / TypeScript MySQL 8.0 JANコードスキャン 楽天商品API マルチテナント Docker Compose GitHub CI/CD
technical notes

JANコードスキャン時に楽天商品APIと連携し、商品名・画像・価格を自動取得。未登録商品でも即座に商品マスタへ登録できる。API取得失敗時も統一した自動下書き作成フローに落ちるよう設計した。

マルチテナント構成で複数店舗を一元管理。テナントごとにデータを完全分離し、テナントオーナー・スタッフ・システム管理者の3ロールで権限を制御している。

GitHub ActionsによるCI/CDパイプラインを構築し、developブランチへのマージで自動デプロイが走る体制を整えた。
