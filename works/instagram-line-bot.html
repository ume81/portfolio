AI LINE Bot // 01

インスタ自動投稿
LINE Bot

LINEで画像と一言を送るだけで、Claude APIがプロ品質のキャプション・ハッシュタグを生成しInstagramへ自動投稿。小規模店舗のSNS運用コストをゼロにする多店舗対応AI投稿支援システム。

demo / screenshot
overview
PROBLEM
「毎日インスタを更新したいが、キャプションを考える時間がない」という飲食店・美容院などの小規模店舗オーナーの課題。専用アプリの導入・操作習得もハードルが高い
SOLUTION
使い慣れたLINEで画像と一言を送るだけで完結。Claude APIが業種・店舗に合わせたプロ品質の投稿文を生成し、確認OKを返すだけでInstagramへ自動投稿
TARGET USER
飲食店・美容院・小売店など小規模店舗のオーナー・スタッフ。PINコード認証で複数スタッフが同じLINEアカウントから投稿できるマルチスタッフ設計
PLATFORM
LINE Bot（主要UI）+ Web管理画面（補助）。バックエンドはXserverでLaravel APIを稼働。店舗ごとに独立したLINE公式アカウントで運用
features
💬
LINEで完結する投稿フロー
画像送信→説明入力→AI生成→確認→Instagram投稿の全フローがLINEチャット上で完結。新しいアプリのインストール不要で、スタッフへの操作説明コストもゼロ。
🤖
3層プロンプトによるAI生成
Claude APIへの指示はベース・業種別・店舗別の3層構造。業種（飲食/美容など）と店舗固有の特徴・トーンを重ねることで、汎用的でなく「その店らしい」投稿文を生成。投稿タイプ（通常・PR・イベント・季節）ごとにプリセットを切り替え。
🔐
PINコード認証・マルチスタッフ
店舗共有PINコードで友達追加したスタッフを認証。初回登録者がオーナー、以降はスタッフとして役割自動付与。複数スタッフが同じLINE公式アカウントを使って投稿できる。
🏪
マルチ店舗対応
店舗ごとに独立したLINE公式アカウント・Instagram連携・プロンプト設定を管理。Webhookは `?store={slug}` パラメータで振り分け。1つのシステムで複数店舗を横断管理できる。
🖥️
Web管理画面
Inertia.js + React製の管理画面で投稿履歴・下書き管理・店舗設定を操作。プロンプトテンプレートのカスタマイズや、LINE/Instagram連携設定もGUIから変更可能。
🚀
Web編集画面 & Instagram自動投稿
LINEの「投稿を確認する」ボタンからWeb編集画面へ遷移。画像のトリミング（正方形・縦長・横長）、AIキャプションの微調整、ハッシュタグ編集ができ、「選択先に投稿」でInstagram Graph API経由で即時投稿。画像はAWS S3に保存し署名付きURLでInstagramに渡す。
📍
Googleビジネスプロフィール連携 （実装予定）
Instagramに加えてGoogleビジネスプロフィールへの同時投稿にも対応予定。店舗の最新情報・お知らせをGoogle Maps上の口コミ欄にも自動展開し、集客チャネルをさらに拡大する。
tech stack
Laravel 10.x / PHP Claude API（Sonnet 4.5） LINE Messaging API Instagram Graph API AWS S3 React / Inertia.js Tailwind CSS MySQL 8.0 MinIO（開発用S3互換） Docker Compose Xserver
technical notes

LINE Webhookのタイムアウト制限（1秒）に対応するため、AI生成処理は非同期で実行しキュー経由で処理。Xserverなどキューワーカーが使えない環境ではCron+データベースキューで代替するデュアルモード設計にした。完了後はプッシュメッセージで結果を返す。

Instagram Graph APIの仕様上、投稿は「メディアオブジェクト作成→公開」の2ステップが必要。画像URLに一時的にアクセス可能なS3署名付きURLを使用し、Instagram側のメディア取得を担保している。

AIプロンプトはベース（全業種共通）→業種別（飲食・美容など）→店舗別（個店のトーン・NGワード）の3層で構成。店舗オーナーがWeb管理画面からプロンプトテンプレートを自由にカスタマイズでき、投稿文のブランドトーンを細かく制御できる。
