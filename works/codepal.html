VSCode Extension // 01

CodePal

VSCodeのパネル内にLive2Dキャラクターが常駐するAIコードレビューアシスタント。コミット時の自動レビューやGitHub連携を、キャラクターとの会話で操作できる開発支援ツール。

demo / screenshot
overview
PROBLEM
コードレビューやGitHub操作はエディタを離れて行うことが多く、コンテキストの切り替えがストレスになる
SOLUTION
VSCode内にAIアシスタントを常駐させ、コードレビュー・タスク管理・PR作成・Issue確認をエディタから離れずに実行できる
TARGET USER
AIツールを日常的に使う個人開発者・エンジニア。キャラクターとのインタラクションで開発体験を豊かにしたい人
PLATFORM
VSCode拡張機能（Windows / Mac / Linux 対応）
features
🤖
Live2Dキャラクター常駐
パネル内にLive2Dキャラクターが常駐。後輩・ツンデレ・執事・メイドなど8種類の性格タイプから選択でき、カスタムキャラクターの作成にも対応。
🔍
自動コードレビュー
コミット時に自動でコードレビューを実行。選択範囲・ファイル単位でのレビューにも対応。基本・詳細の2モードで詳細度を切り替えられる。
🌐
マルチAIプロバイダー対応
Gemini / Claude / OpenAI の3プロバイダーに対応。用途やコストに合わせてモデルを切り替えられる。Gemma（オープンソース）にも対応。
🐙
GitHub連携
GitHub Issues の確認・AIによる自動解決提案・PRの自動作成をエディタ内から実行。Octokit経由でGitHub APIと連携。
✅
タスク管理
キャラクターとの会話でタスクを追加・確認。コーディング中でも画面を切り替えずにタスクを管理できる。
tech stack
TypeScript VSCode Extension API Live2D（pixi-live2d-display / Pixi.js） Gemini API Claude API OpenAI API Octokit（GitHub API） simple-git Webpack
technical notes

VSCode WebView APIを使いパネル内でLive2DキャラクターをPixi.jsでレンダリング。拡張機能のコンテキストとWebView間のメッセージパッシングで、AIレビュー結果やGitHub情報をリアルタイムに反映している。

simple-gitでコミット差分を取得しAI APIに渡すことで、コミット内容を自動解析したレビューを実現。Gemini / Claude / OpenAI のプロバイダーを設定で切り替えられるよう抽象化した。

キャラクターの性格タイプごとにシステムプロンプトを定義し、レビュー文体・アイドルメッセージ・インタラクションメッセージを変化させることで、キャラクターとしての一貫性を保った。
