デスクトップマスコット
タスク管理

デスクトップに常駐するマスコットキャラクターがAIアシスタントとして業務タスクを管理するElectronアプリ。メール内容からのAIタスク自動生成・チャット形式でのタスク追加に対応。

demo / screenshot
overview
PROBLEM
業務メールから手動でタスクを起こす作業が手間。タスク管理ツールを別途開くのが面倒で、タスクの記録漏れが起きやすい
SOLUTION
IMAPでメールを直接取得しAIがタスクを自動生成。デスクトップ常駐のキャラクターにチャット感覚でタスクを追加でき、Google認証によるマルチデバイス同期にも対応
TARGET USER
メールベースで業務が動くビジネスパーソン。複数PC間でタスクを共有したい人
PLATFORM
Windowsデスクトップアプリ（Electron）。ローカルはSQLite、クラウド同期はLaravel API + Google OAuth
features
📧
IMAPメール取得 + AIタスク自動生成
IMAPで受信メールを直接取得し、Gemini / OpenAI APIがタスクを自動抽出・登録。対応期限・優先度の提案まで行う。
💬
チャット形式でタスク追加
マスコットキャラクターとの会話形式でタスクを追加・確認・完了。自然な言葉でタスクを登録でき、フォーム入力より素早く操作できる。
🖥️
デスクトップ常駐
他のアプリを使用中でも常にデスクトップに表示。アプリを切り替えることなくタスクの確認・追加が可能。
🔗
Google OAuth + マルチデバイス同期
Googleアカウントでログインし、Laravel API経由でタスクをクラウド保存。複数PCからデータを共有できる。
🔔
キャラクター吹き出し通知
新着メール・今日のタスク一覧・期限30分前アラートなど、キャラクターとの会話形式で吹き出し通知が届く。作業中でも自然に情報を受け取れる。音声読み上げは追加予定。
tech stack
Electron React / TypeScript Vite Gemini API OpenAI API SQLite（better-sqlite3） Laravel Socialite（Google OAuth） Laravel 12 / PHP 8.2 Laravel Sanctum imapflow / mailparser Tiptap dnd-kit recharts Tailwind CSS
technical notes

ElectronのMainプロセスとRendererプロセスをIPC通信で接続する構成。RendererからDB・OS APIに直接アクセスせず、必ずMainプロセス経由にすることでセキュリティと責務の分離を徹底している。

imapflow / mailparserでIMAPサーバーから受信メールを直接取得。メール本文をGemini / OpenAI APIに渡してタスクを自動抽出する。オフライン時はSQLiteでローカル管理し、オンライン時はLaravel API + Laravel Sanctum経由でクラウド同期する設計。

Google OAuth（Laravel Socialite）によるログインでマルチデバイス対応を実現。CodePal（VSCode拡張）・オーバーレイAIアシスタント（Android）と合わせ、開発・業務・日常の3シーンでAIタスク管理を実現するシリーズ作品。
